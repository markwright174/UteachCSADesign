Day 2 (of 14)

What is Computer Science? (25 min.)

  1. Direct students to the Introduction to Computer Science student page, section What is Computer Science?
  2. Working in their groups, direct students to answer the following questions:
    • What is computer science? T he science that deals with the theory and methods of processing information in digital computers, the design of computer hardware and software, and the applications of computers. (dictionary.com)
    • Why is studying computer science important? Using computer science to solve problems allows society to automate many processes that would otherwise take much more time to accomplish.
      After giving students some time to discuss, have groups formulate, and share their answers to each of the questions. Discuss as a class the suggested answers.
  3. Direct each group to answer the following:
    • What is a computer? A n electronic device for storing and processing data, typically in binary form, according to instructions given to it in a variable program. (dictionary.com)
    • What is program code? T he set of instructions forming a computer program, which is executed by a computer. (wikipedia.com)
    • What is the difference between computer hardware and software?  Computer hardware refers to the physical parts of a computer and related devices such as monitors, keyboards, mice, printers, scanners, motherboards, hard drives, and RAM. You can touch and feel the hardware components. (techterms.com)
      Computer software refers to programs and applications that run on a computer. You cannot touch or feel software. Instead, you use a computer to execute and use software for various purposes.       (techterms.com)
      Ask groups to share their answers and discuss answers as a whole class.
  4. Direct students to read the section Problem Solving.
  5. Discuss what problems have been solved by incorporating computer science into the solution. Let students know that when we use the term “problem” in computer science, we are not referring to a situation in which something is wrong. Typically, a “problem” refers to a task that needs to be completed.
  6. Direct students to use the definition of computing innovation as given by the College Board for the APCS Principles course, A computing innovation is an innovation that includes a computer or program code as an integral part of its functionality, and discuss in their groups the following:
    • Name some computing innovations that have affected your life.
    • Indicate whether the innovation includes a computer or program code as an integral part of its functionality.
    • Name the problem that this computing innovation solves. 
    • Name a beneficial effect and a harmful effect of the computing innovation.
      Ask groups to share their answers.
  7. Discuss as a class the legal issues and intellectual property concerns surrounding the development and creation of programs and software. 
    • Would software apps fall under intellectual property?
    • If you work for a company and develop an app, who owns the code, you or the company?
    • Is it legal to use other people's code that you find on websites such as Stack Overflow or Github?
    • Let's say you had a great idea for an app and told a friend about your idea. A few years later, you see an app in the app store based on your idea. After some investigation, you discover your friend wrote the app. You never had the time to write the app yourself. Did your friend steal your intellectual property?
  8. Direct students to think about the following questions and share within their groups:
    • Why is studying computer science important to you? Why are you taking this course?

 

Programming Languages (25 min.)

  1. Direct students to the Programming Languages section.
  2. Introduce a discussion regarding programming languages by having groups answer the following:
    • What is a programming language? A set of commands, instructions, and other syntax used to create a software program. (techterms.com)
    • Define low-level programming language. A programming language that uses machine code that a computer can understand for communicating instruction.
    • Define high-level programming language. A programming language that uses English-like statements and commands that are compiled into machine code that the computer can understand and execute.
  3. Direct each group to research the history of programming languages and create an infographic that shows a timeline of programming languages from earliest to latest.
    Assignment requirements:
    • Include at least 8-10 programming languages in your timeline.
    • Indicate if the language is low-level or high-level.



Day 3 (of 14)

Introduction to AP Computer Science A (25 min.)

  1. Direct students to read Introduction to AP CSA.
  2. Students will go to the link to the AP Computer Science A Course and Exam Description document from the College Board.
  3. Direct students to find the page titled "About the AP Computer Science A Course"  and read the introductory paragraph.
    “AP Computer Science A introduces students to computer science through programming. Fundamental topics in this course include the design of solutions to problems, the use of data structures to organize large sets of data, the development and implementation of algorithms to process data and discover new information, the analysis of potential solutions, and the ethical and social implications of computing systems. The course emphasizes object-oriented and design using the Java programming language. "
  4. Discuss the section College Course Equivalent.
    "AP Computer Science A is equivalent to a first-semester, college-level course in computer science."
  5. Discuss the Computer Language required for the course.
    "The AP Computer Science A course requires that solutions of problems be written in the Java programming language. Because the Java programming language is extensive, with far more features than could be covered in a single introductory course, the AP Computer Science A Exam covers a subset of Java."
  6. Direct students to find and read through the section under the "Introduction" page that describes the goals of an introductory, college-level computer science programming course. Note that these are the Computational Thinking Practices as outlined in the Course Framework.

    "The AP Computer Science A course reflects what computer science teachers, professors, and researchers have indicated are the main goals of an introductory, college-level computer science programming course:"

      • Program Design and Algorithm Development - Determine required code segments to produce a given output.
      • Code Logic - Determine the output, value, or result of given program code given initial values.
      • Code Implementation - Write and implement program code. 
      • Code Testing - Analyze program code for correctness, equivalence, and errors. 
      • Documentation - Describe the behavior and conditions that produce the specified results in a program.
      • Ethical Computing - Understand the ethical and social implications of computer use.

  7. One of the overarching concepts in this course is object-oriented programming. Since Java forces developers to use object-oriented design when developing programs, it is the language of choice by the College Board.
  8. Direct students to conduct a Google search to answer the following:
    • How popular is Java as a programming language?
    • Name some well-known applications that are written in Java.
    • Research the terms “platform independent” and “language portability” and describe how Java fits into this feature.

A programming language is considered platform-independent or portable if any program created in that language on one computer or operating system will work on another computer or operating system with minimal changes.
Java uses a compiler to translate the program source code (.java file created by the programmer) into bytecode (.classfile). Java then continues and uses an interpreter (Java Virtual Machine, JVM) to translate the bytecode into machine code line by line during execution.

Day 4 (of 14)

My First Python Program  (30 min.)

  1. Direct students to the My First Python Program student page and provide the link to the My First Python Program in Repl.it.
  2. Under Analyzing Code , ask the students to read the Python code and think about what it does. After giving time for students to read the code, ask for student volunteers to share their ideas of what the program does.

  3. Direct students to follow the instructions under Writing Code to create a new Repl program in Python (with Turtle) and type the code as given. Answer any questions that arise.
  4. Direct students to execute the code by pressing the run button as instructed.
  5. If students have errors, direct them to the next section, Syntax Errors . Students should try to determine their errors on their own and then seek help from a classmate or the instructor if needed.
  6. Once the program executes successfully, students should change the values of score for different outputs.
  7. Students should read the section, Syntax Errors , and experiment with creating these different kinds of common syntax errors and correcting them.
  8. Be available to provide assistance for students. Students can also help each other if needed.

 

Explore Repl.it (20 min.)

  1. Direct students to explore Repl.it by clicking on my repls.

  2. Students should see a list of their programs. To edit or continue working on a program, they click the name of the program to open it.
