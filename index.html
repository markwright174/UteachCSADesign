FAQ
How is the curriculum structured?

The UTeach CSA curriculum consists of seven units, each of which is centered around a unit project. Projects are introduced at the beginning of every unit through an anchor video. The course is narrative-driven, with students following a diverse cast of characters who travel across the globe and solve real-world problems along the way. 

What are the technology requirements of the course?
Technical Specifications
Repl.it is used as the IDE for this course, allowing students to program, compile, and run Java programs all in one convenient online platform.
Java is the primary programming language used in this course, following the College Board and AP testing requirements.
Python with Turtle is the programming language students use to create personal avatars at the beginning of the course.

Why does the course start in Python, rather than Java?
Students briefly use the Python with Turtle programming language to create personal avatars in Unit 1. We have found that Python provides a quicker on-ramp into coding than Java does, and using Python helps build students' confidence at the beginning of the course. In Unit 1, students master basic tools and concepts through Python, such as algorithms and abstraction, which are applied in later units using Java.
Is this course approved by the College Board?
All UTeach CSA curriculum units are aligned to the updated College Board AP CSA Course Framework for 2020.  Because UTeach CSA is officially endorsed by the College Board, teachers can select the UTeach CSA syllabus for their course audit. For more detailed information on course alignment, please refer to the pacing guides at the beginning of each unit. Note that our unit structure differs slightly from the College Board's because some of our units combine multiple units from the course framework. UTeach CSA lesson plans are aligned to the AP Classroom and direct teachers to appropriate Personal Progress Checks throughout the course.
